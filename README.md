# Heart Disease  System

## Objective

Build a recommendation system using a heart disease dataset to analyze health parameters and provide insights that help identify individuals at risk of heart disease.

## Tools Used

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

## Steps Performed

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Correlation Analysis
4. Data Visualization
5. Building Recommendation Logic
6. Insights and Conclusion

## Dataset

Heart disease dataset containing medical attributes such as age, cholesterol level, blood pressure, chest pain type, and heart rate.

## Key Insights

* Age and cholesterol levels show strong influence on heart disease risk.
* Higher blood pressure is often associated with increased risk.
* Certain chest pain types are strong indicators of heart conditions.
* Combining multiple health indicators improves recommendation accuracy.
